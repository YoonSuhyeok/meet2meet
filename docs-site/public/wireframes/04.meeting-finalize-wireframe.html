Edge Cases and Policy
E1
동률 후보 다수 -> 주최자 수동 선택 필수, 자동 확정 금지
E2
확정 후 재투표 요청 -> 상태를 reopened로 바꾸기 전에는 불가
E3
알림 발송 실패 -> 확정은 유지, 발송 재시도 큐 안내
E4
확정 API 중복 호출 -> 409 응답과 함께 현재 확정 슬롯 반환
E5
결과 링크 만료 -> 404 + 재초대 요청 CTA
