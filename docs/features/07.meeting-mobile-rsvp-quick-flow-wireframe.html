5
Edge Cases and Policy
E1
초대 만료 링크 -> 410 응답, 만료 안내와 재초대 요청 CTA 표시
E2
중복 제출 -> 동일 status 재제출 시 200 idempotent 처리
E3
네트워크 불안정 -> 로컬 큐 1회 저장 후 재시도 버튼 노출
E4
Undo 기간 초과 -> undo API는 410, 새 응답 제출로만 변경 허용
E5
미응답자 과다 -> 임계치(25%) 이상 시 호스트 보드에 재촉 CTA 강조
