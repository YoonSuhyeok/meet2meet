Edge Cases & Policy
E1
이미 로그인된 상태에서 /login 접근 → /로 리다이렉트
E2
JWT 만료 → GET /me 401 → 세션 쿠키 삭제 → 로그인 페이지로
E3
OAuth 제공자 거부 (사용자 취소) → /login?error=auth_failed
E4
지원하지 않는 provider → /login?error=auth_failed
E5
동일 이메일 다른 제공자 → 별도 계정 (추후 연동 고려)
