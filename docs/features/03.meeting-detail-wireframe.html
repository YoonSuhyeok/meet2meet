5
Edge Cases & Policy
E1
미팅 없음/삭제됨 → 404 화면 + 홈으로 이동 버튼
E2
투표 마감 후 제출 → 제출 차단 + "마감된 미팅" 배너
E3
동일 이름 재응답 → 기존 응답 업데이트 확인 모달
E4
네트워크 끊김 → 로컬 임시 보관 후 재시도
E5
슬롯 미선택 제출 → 버튼 비활성화 + 안내 문구
